# Unified Chat Aggregator

Memoria operacional do projeto.

- O entregavel principal e um video de demo mostrando o app desktop funcionando.

- Validar primeiro com mock, depois uma plataforma real ponta a ponta por vez.
- X e Kick tem fragilidade maior por dependerem de caminhos nao-oficiais para leitura.
